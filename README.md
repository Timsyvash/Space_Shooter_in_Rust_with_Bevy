# Space_Shooter_in_Rust_with_Bevy
Я зробив гру Space Shooter на Rust з Bevy
# Про Space Shooter
Space Shooter (Космічний шутер) — це жанр аркадних ігор, де гравець керує космічним кораблем, знищуючи хвилі інопланетних ворогів та босів, щоб врятувати галактику. Зазвичай це динамічні «стрілялки» (shoot 'em up) з вертикальною або горизонтальною прокруткою, що вимагають спритності для ухилення від куль. 
Основні особливості Space Shooter:
-Геймплей: Управління здійснюється одним пальцем (на мобільних) або мишею, корабель стріляє автоматично або по команді, основна мета — ухилятися від ворожого вогню.
-Прогресія: Зароблені монети та кристали витрачаються на покращення зброї, щитів та купівлю нових, потужніших кораблів.
-Рівні та боси: Гра складається з сотень рівнів, які закінчуються битвами з гігантськими босами.
-Безкоштовність: Більшість таких ігор на Android, як-от Space Shooter: Galaxy Attack, є безкоштовними та працюють без інтернету.
Це ідеальний варіант для швидкого зняття стресу, де поєднуються класичні елементи ретро-ігор (як Galaga) із сучасною графікою та візуальними ефектами.
# Про Rust
Rust — це сучасна компільована мова програмування загального призначення, орієнтована на безпеку, швидкість і паралелізм. Вона забезпечує високу продуктивність рівня C/C++, уникаючи поширених помилок роботи з пам'яттю (segfaults, витоки) завдяки унікальній системі володіння (ownership) без збирача сміття. 
Де використовують Rust:
Системне програмування: Розробка операційних систем (наприклад, Redox), браузерних рушіїв (Firefox, Servo).
Вбудовані системи (Embedded) та IoT: Завдяки низькорівневому доступу до обладнання.
Високонавантажені вебсервіси: Завдяки швидкості та паралелізму.
WebAssembly: Швидка робота в браузері.
Rust стабільно займає перші місця у рейтингах найулюбленіших мов програмування, згідно з опитуваннями Stack Overflow. Він розробляється спільнотою та підтримується Rust Foundation
# Про Bevy
Bevy — це сучасний, високопродуктивний ігровий рушій з відкритим кодом, написаний мовою Rust. Він базується на архітектурі ECS (Entity-Component-System), що забезпечує гнучкість, безпеку та високу швидкість роботи. Рушій активно розвивається (версія 1.0 очікується/актуальна у 2025 році), пропонуючи зручні інструменти для розробки.
Ключові особливості Bevy:
-Мова програмування: Написаний на Rust, що гарантує безпеку пам'яті та високу продуктивність.
-Архітектура ECS: Орієнтований на дані підхід (Entity-Component-System) дозволяє ефективно керувати ігровими об'єктами.
-Відкритий код: Безкоштовний та розвивається спільнотою.
-Модульність: Легко розширюваний завдяки плагінам.
-Перспективи: Вважається одним з найкращих ігрових рушіїв для інді-розробників у 2025 році.
Переваги:
-Висока продуктивність: Завдяки ECS та Rust.
-Безпека: Гарантована мовою Rust.
-Сучасний підхід: Розробка орієнтована на дані.
Недоліки:
-Час компіляції: Іноді може бути проблемою (інкрементальна компіляція).
-Bevy підходить для створення як 2D, так і 3D ігор.
